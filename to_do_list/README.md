# To do list

Proyecto creado con [Create React App](https://github.com/facebook/create-react-app).

## Descripción

To do list es una aplicación multiplataforma para anotar un listado de tareas pendientes, que podrás tachar y eliminar, las siguientes instrucciones te van a permitir tener una copia de este proyecto para fines de desarrollo y pruebas.

## Vamos!!

### Instalación 🪛
Este comando te permitirá tener las dependencias de la aplicación.
```
npm install
```
Con este comando tendrás en tu browser corriendo la aplicación ¡disfrútala!
```
npm start
```
